# Semantic SEO

Project for the **Web and Data Science** course at **Politecnico di Milano** (A.Y. 2025/2026).

This repository contains a Semantic SEO project based on the website of the Municipality of Grosotto.  
The goal of the project is to analyse the website, identify its main entities, and enrich selected pages with structured data using **Schema.org** and **JSON-LD** markup.

The project focuses on improving the semantic description of the website content, making entities such as places, services, events, FAQs, and organizations more explicit and machine-readable.

## Project Description

The original website already included some automatically generated structured data. However, the markup was generic and did not describe the specific entities of the website in enough detail.

For this reason, the autogenerated markup was commented out and replaced with custom JSON-LD structured data. The new markup was designed to better represent the content of the selected pages and to include relevant information such as:

- public places and municipal offices;
- services offered by the municipality;
- events;
- frequently asked questions;
- opening hours, addresses, contacts, and other entity-specific properties.

The structured data was implemented using Schema.org types and validated with Google tools for structured data testing.
